# Core IP Networking Concepts

1. **What is an IP address?**
   - Learn the definition, purpose, and format of an IP address.

2. **What are IP address classes?**
   - Understand Class A, B, C, D, E, their ranges, and usage.

3. **What is a subnet mask?**
   - Learn how subnet masks work, why they’re needed, and how they relate to IPs.

4. **What is a public & private IP?**
   - Know the difference, usage, and the private IP ranges.

5. **What is NAT (Network Address Translation)?**
   - Understand what NAT is, why it’s used (especially with private/public IPs), and basic operation.

6. **What is IPv4 & IPv6?**
   - Learn the differences, address formats, and why IPv6 was introduced.

# Device Communication and Network Components

1. **What are the 4 layers of the TCP/IP model?**
   - Know the names and simple function of Application, Transport, Internet, and Link layers.

2. **What is a switch?**
   - Learn what a switch does in a local network and how it forwards data.

3. **What is a router?**
   - Understand what a router does, how it connects networks, and how it differs from a switch.

4. **What is a modem?**
   - Know what a modem does and its role in connecting a home network to the internet.

5. **What is a loopback address?**
   - Learn what a loopback address is, its common value, and why it’s useful.

6. **What are private IP ranges?**
   - Know which IP ranges are private and why we use private IP addresses in local networks.
